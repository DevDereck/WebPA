# ChurchForm

Sitio web moderno en tonos turquesa para la Iglesia Cristiana Piedra Angular.

## Archivos
- `index.html`: página principal con secciones de inicio, sobre nosotros, ministerios, contacto y dirección.
- `style.css`: tema unificado con gradientes turquesa, animaciones `fadeIn`, tarjetas y estilos de formulario/botones.
- `assets/logo.svg`: logo vectorial inspirado en el arte proporcionado.

## Uso
1. Abre `index.html` en tu navegador o publícalo en tu hosting estático favorito.
2. Sustituye textos, horarios y medios de contacto según necesites.
3. Reemplaza el bloque de "Mapa interactivo" por tu mapa embebido (ej. Google Maps iframe).

## Personalización rápida
- Cambia la paleta ajustando las variables en `:root` de `style.css`.
- Ajusta la navegación o secciones editando el marcado dentro de `index.html`.
- Conecta el formulario a tu backend o servicio de formularios preferido.
